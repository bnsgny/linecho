# linecho
echoするだけの簡単なline bot

# 必要なもの
開発者登録とかアプリ登録は各自でやってくれ

herokuで動かすためにFixieっていうIP固定サービス使ってるけどよしなに

## env

| name | description|
|------|------------|
| CHANNEL\_ID     | channel id     |
| CHANNEL\_SECRET | channel secret |
| MID            | bot mid        |
